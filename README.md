# 颜色格式转换

> HEX / RGB / HSL / HSV / CMYK 五向互转，实时预览，色值调试辅助

## 功能概览

- **HEX / RGB / HSL / HSV / CMYK 五种颜色格式实时互转**
- **颜色拾取器：点击色块或输入色值自动识别格式**
- **调色板预览：主色 + 类似色 / 互补色 / 三分色自动生成**
- **CSS 色值输出：直接复制标准 CSS 颜色代码**
- **颜色命名提示：相近的 Web 安全色名称推荐**

## 操作说明

1. 打开 index.html（浏览器直接运行，无需服务器）
2. 按界面引导输入/操作
3. 结果实时预览，支持一键复制导出

## 技术栈

- **HTML5**：语义化结构
- **CSS3**：白色极简主题，Flex/Grid 布局
- **Vanilla JavaScript**：纯原生代码，零依赖
- **ES6+**：sync/await、箭头函数、模板字符串
- **localStorage**：偏好设置持久化
